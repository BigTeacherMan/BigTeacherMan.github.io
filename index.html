My Work

Python

Chambers

This is a game I started developing in October of 2023. I had a student ask if it's possible to create a Russian Roulette video game. Kind of an odd inquiry, but this was my introductory Python class. I took this as a challenge and I started coding a Russian Roulette game that eventually garnered the name: "Chambers". This is certainly a work in progress, but it has taught me much about how to code better in Python and what to expect when working on a program that will be exported. I have also been fortunate to include one of my old students on this project. He, Issac Robertson, has done all of the artwork for this project and has done a tremendous job. This has given me the chance to oversee a project and work with others to get it done.

C#

Coding Challenge

This was my first foray into the realm of c#. What was described as "easy" and "simple" became an overwhelming goliath of a challenge as I realized my Python mind only knew so much when it came to this unfamiliar coding language. New keywords, modules/library, syntax, and including a semicolon and the end of every line of code was frustrating at the least, but it was envigorating at the most. Taking on this coding challenge reignited my desire to learn how to code and c# is my new target. This code is certainly going to look like it came from the hand of a beginner, but I will get better.
